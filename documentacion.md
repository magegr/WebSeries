# Documentación de la Web de Series

## 1. Descripción del Proyecto
Aplicación web de cine que permite ver información sobre **películas, series y actores** usando la API de **The Movie Database (TMDB)**. 

Funcionalidades principales:
- Listado de series por popularidad.
- Detalles de cada serie (sinopsis, temporada, episodios, actores).
- Sistema de búsqued.

---

## Requisitos

Antes de ejecutar la aplicación, necesitas tener instalado:

- [Node.js](https://nodejs.org/)
- npm

Puedes comprobarlo con:

```bash
node -v
npm -v
```
>[!IMPORTANT] ## Instalacion
Clonas el proyecto e instalas dependencias:
```bash
npm install
```
